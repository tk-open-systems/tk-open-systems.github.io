Microchip/Atmel SAMA5D2 Linux Board Bring-Up Completed

Projects / By Jonathan Ben-Avraham / Leave a Comment / October 3, 2018

TkOS has successfully completed a board bring-up project for a major automotive sector manufacturer. The board was customer designed, based on the SAMA5D2 reference pictured. We built a mainline Linux kernel for the board and a root filesystem using Buildroot, and made video driver customizations.
